K-Pop Idol Face Identifier

An app built with Microsoft Cognitive Services Face (Bing Search and Face), Angular.JS, Node.JS, Express.JS, and HTML/CSS. It trains the system to identify Korean Pop idols faces with 30 images per each celebrity. Currently, there are 78 celebrities available to be identified.
